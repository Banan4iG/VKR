Файл maps.py берёт растровое изображение(*_set.png) по координатам указанном в файле coords.txt, оставляет только здания(*_bld.png) и добавляет рамку в 20 px (*_contr.png)
Файл createLayer.py создаёт векторный слой из файла (*_contr.png) полученного в файле maps.py
Файл Nir.py сопоставляет растровую и векторную карты( мультяшная веторная карта полученная из файла createLayer.py)
Файл Nir2.py работает на отрисованных вручную векторных объекта( test5.shp)
Test5 нужно переместить в слои и поменять систему координат на Gauss-Kruger zone 8 для отображения объектов
